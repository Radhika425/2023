expt 7 nmap

Perform network discovery using nmap

Commands:

1. nmap google.com

2. Nmap google.com facebook.com

3. (HOSTNAME idemtify) Sudo nmap-sL google.com

4. (-SS flag is used for TCP SYN Scan) Sudo nmap -sS google.com

5. Nmap -A google.com

6. Sudo nmap -O google.com

7. ("-sU" flag is used with nmap to perform a UDP scan) Sudo nmap -sU google.com

8. (Firewall settings) Sudo nmap -sA google.com
